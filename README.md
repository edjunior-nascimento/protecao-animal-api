# protecao-animal

Site criado para ONG de Proteção de Animais, com o intuito de melhorar a experiencia do usuario, gerenciando doações, adoções, catrações e venda de rifas

# Tecnologias utilizadas
-----------
- Java 17
- Spring Framework 3
- Mysql 8.0.32
